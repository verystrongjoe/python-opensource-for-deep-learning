- 파이선 3.6이 나온지 10년이나 흘렀다고, 데이터 분석과 딥러닝을 위해서는 64비트로 가는게 맞음.
- 파이선 아나콘도 설치를 할경우 환경변수 추가하는 부분은 제외하자( 기존의 파이선 바이너리 셋팅의 혼선을 줄이기 위해)
